# Bike Sales Analysis

This project analyzes bike sales data using Python to identify customer behavior, product performance, pricing patterns, seasonal trends, and market expansion opportunities.

## Project Objectives

- Analyze customer purchasing behavior
- Identify high-performing products and categories
- Evaluate pricing and profitability relationships
- Detect seasonal sales trends
- Explore international market opportunities
- Segment customers using K-Means clustering

## Methods Used

- Exploratory Data Analysis (EDA)
- K-Means Clustering
- Elbow Method
- Silhouette Score
- Correlation Analysis
- Time Series Analysis
- Data Visualization

## Key Findings

- Young Adults (25–34) generated the highest average revenue and profit
- Road Bikes were the strongest revenue-generating product category
- Premium-priced products contributed significantly to profitability
- Sales peaked during May–June and December
- The United States was the largest overall market
- Four distinct customer segments were identified using clustering analysis
